// HOW IT WORKS 5-step inference pipeline
// audio → preprocessing → spectrogram → CNN → weighted prediction
STEP_01
File type check and conversion to .wav format for standardized processing.
STEP_02
Audio resampled to 22050 Hz. Metadata extracted: length, pitch, average frequency.
STEP_03
Noise reduction applied to amplify important bird-call frequencies and suppress background.
STEP_04
Mel-spectrogram generated for every 5-second window of audio, producing an array of image segments.
STEP_05
CNN predicts on each spectrogram. Weighted final prediction combines all segment confidences.
