Overview

Welcome to my personal website! Here, I showcase my projects and share insights into my academic journey. I am an Honors Computer Science student at The University of Alabama at Birmingham, expecting to graduate in Spring 2025.

Waste Detection Drone
Research!

Utilized machine vision AI to program a drone for waste detection, specifically designed for government waste management. The solution identifies high-concentration waste areas using the YOLOv5 algorithm from Ultralytics, trained with custom annotated data via Roboflow.io.

This project is currently undergoing academic research at UAB, with a paper scheduled for release in the fall. Additionally, I employed the Tello SDK to enable automated drone flights, integrating built-in logging and a screenshot system.

Auto Viper
Project Lead!

I lead the development of an automated testing framework for Viper Vision software at Viper Imaging LLC. This framework efficiently conducts comprehensive tests, significantly reducing integration, testing, build, and release times.

The framework integrates various design patterns and includes features such as ROI state logging, database access, screen manipulation, and dynamic camera management. It is implemented in Python, leveraging libraries such as OpenCV.

Viper Vision
Contribution!

Working with the team at Viper Imaging LLC on developing and updating our software, Viper Vision. Viper Vision is a thermal camera hub where you can connect cameras and get data and live image feed from them, as well as numerous other actions. Developed on thee .NET framework using C#.

Some of my tasks have been developing new features for newer thermal cameras with features including accurate thermal scale display, web streaming, range selection, and mirror and rotate calculations.
