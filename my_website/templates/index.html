Viper Vision Automation

I am in charge of developing an automated testing framework for the Viper Vision software developed by Viper Imaging LLC. This framework is responsible for conducting thorough tests to cut back on integration, testing, build, and release times.

Framework incorporates various design patterns and features such as ROI state logging, database access, screen manipulation, and dynamic camera management. Developed using python and various libraries such as OpenCV.

Active Development!

Viper Vision
Contribution!

Working with the team at Viper Imaging LLC on developing and updating our software, Viper Vision. Viper Vision is a thermal camera hub where you can connect cameras and get data and live image feed from them, as well as numerous other actions. Developed on thee .NET framework using C#.

Some of my tasks have been developing new features for newer thermal cameras with features including accurate thermal scale display, web streaming, range selection, and mirror and rotate calculations. Other tasks I have contributed to were live chart database separation, UI changes, and various others
