VisAudio

VisAudio is a desktop app that I made for myself to deal with audio file manipulation, and YouTube audio downloading for all of my projects I deal with. I also added mel-spectrogram and waveform graphs for some audio visualization, with a visual EQ generator currently being worked on.

Some features include file type conversion, bit-rate resampling, noise reduction, and visual graph downloads. Along with file manipulation, it also includes a YouTube downloader where you can input any valid link to get that video's audio. It could do the video but... this is an audio themed application.

File types that are supported are: mp3, wav, flac, ogg, m4a, and aac with a more features planned for the future

FretWatch
In dev

FretWatch is an OpenCV and PyTorch project centered around live guitar note documentation. FretWatch will be able to take in video or live feed and audio and will document the notes and chords played.

This project uses various signal processing techniques to extract data in a meaningful way. Using librosa as an audio signal processor and opencv for image manipulation, I can use these tools to extract audio features currently to detect certain notes played.

Born in Spellbooks 1.20.1
Alpha

Born In Spellbooks is an open source Minecraft forge 1.20.1 mod for Iron's Spells & Spellbooks that I am working on. It is a compatibility mod combining Irons Spellbooks with the mod Born In Chaos. I used decompiling methods to learn the project structure of both mods and with this knowledge I am making a mod combining the two. The project is in alpha with around 17 spells coded in with three releases so far, but none are public yet. This is a community supported mod with other people helping me make textures/models for each spell, along with spell ideas and open source contributions.

This project is heavily focused on learning both codebases and figuring out ways to combine them. With problems such as model rendering, entity sharing, animation layering, and entity/spell/effect/etc registration.
